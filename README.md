Diff Review
===========

A simple tool utilize *quickfix* and [vim-fugitive](https://github.com/tpope/vim-fugitive) to review your code changes.


Prerequisites
--------------

* VIM 8.0
* vim-fugitive

Installation
------------

Place this in your .vimrc:

```
Plugin 'codable/diffreview'
```

Usage
-----

### Start review changes from master

```
Greview master
```

### Move to next file

Type `]q`.

### Move to prev file

Type `[q`.
